# YT-DLP Desktop — 产品设计文档

## 一、产品定位

一款面向普通用户的**轻量级视频下载工具**，为 yt-dlp 提供直观易用的图形界面，
降低命令行使用门槛，让非技术用户也能一键下载在线视频。

## 二、目标用户

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| 普通用户 | 不懂命令行，想下载视频收藏 | 粘贴链接 → 一键下载 |
| 内容创作者 | 需要批量下载素材 | 批量下载、格式选择、字幕提取 |
| 进阶用户 | 了解 yt-dlp，嫌命令行麻烦 | 常用参数可视化 + 自定义命令 |

## 三、设计原则

- **极简优先**：核心流程不超过 3 步
- **渐进披露**：基础功能一目了然，高级功能按需展开
- **透明可控**：下载过程清晰可见，用户始终掌控  

## 四、功能优先级

### P0 核心功能（MVP）
- 单个视频下载
- 格式/画质选择
- 下载进度显示
- 下载历史记录

### P1 重要功能
- 批量下载（多链接/播放列表）
- 字幕下载
- 仅提取音频（MP3）
- 下载路径设置
- 视频信息预览（缩略图/时长/标题）

### P2 增强功能
- 代理设置
- Cookie 导入（登录态视频）
- 自定义文件命名模板
- 剪贴板监控
- yt-dlp 内核自动更新
- 自定义命令行参数

## 五、核心用户流程
- 粘贴链接 → 自动解析（显示预览） → 选择画质（可跳过） → 开始下载 → 完成通知

## 六、界面设计

### 整体布局
- 单窗口 + 三标签页（下载 / 队列 / 设置）
- 自定义无边框标题栏
- 支持系统暗色/亮色主题

### 下载页面
- URL 输入框（支持粘贴自动解析）
- 视频预览卡片（缩略图、标题、时长、上传者）
- 下载选项（格式、画质、字幕、保存路径）
- 大尺寸下载按钮
- 当前下载进度列表

### 队列/历史页面
- 活跃下载任务（进度条、速度、ETA）
- 已完成/失败的历史记录
- 操作按钮（打开文件、打开文件夹、重试、删除）

### 设置页面
- 通用设置（路径、并发数、命名模板）
- 网络设置（代理模式）
- Cookie 设置（文件导入/浏览器读取）
- 高级设置（yt-dlp 版本、额外参数）

## 七、技术架构
┌─────────────────────────────────┐
│ UI 层 (Vue 3) │
│ Pinia 状态管理 + 组件化 │
└──────────────┬──────────────────┘
│ Tauri IPC
┌──────────────▼──────────────────┐
│ 业务逻辑层 (Rust) │
│ 命令处理 / 配置管理 / 数据库 │
└──────────────┬──────────────────┘
│ 进程调用
┌──────────────▼──────────────────┐
│ 核心引擎层 │
│ yt-dlp (视频下载) │
│ ffmpeg (格式合并/转码) │
└─────────────────────────────────┘

### 技术选型
- **前端**：Vue 3 + TypeScript + Pinia
- **后端**：Rust + Tauri 2
- **数据库**：SQLite（历史记录）
- **配置**：JSON 文件

### 优势
- 安装包 ~15MB（vs Electron ~150MB）
- 内存占用 ~30MB（vs Electron ~200MB）
- 原生窗口体验

## 九、关键指标

| 指标 | 目标 |
|------|------|
| 首次下载成功率 | > 95% |
| 操作步骤（单视频） | ≤ 3步 |
